Methodological Framework

Covariance-Based Symmetry Assessment & Exponentiality Testing under Multiplicative Distortion

§ 1
1 Multiplicative Distortion Model
X̃ = ψ(U) · X
X unobservable  ·  U observable confounding  ·  ψ(·) unknown smoothing function  ·  E[ψ(U)] = 1
↓ ↓
Path A · ψ(U) = 1 Path B · ψ(U) ≠ 1
§ 2
2a Direct λ Estimation
X observed directly — four estimators:
λ̂₁ = X̄ (MLE) λ̂₂ = Σlog(Xᵢ)(Xᵢ−X̄)/n λ̂₃ = 1/(2f̄) λ̂₄ = Σ(1−F̂)/f̂/n
λ̂₄ : MLE-level efficiency (Var = λ²) · λ̂₃ , λ̂₄ robust to outliers
§ 3
2b Conditional Mean Calibration
ψ̂(U) = Ê(X̃|U) / X̄̃  →  X̂C,i = X̃i / ψ̂C(Ui)
Nadaraya-Watson kernel estimator
λ̂C,1 = 1/(2f̄C) λ̂C,2 = Σ(1−F̂C)/f̂C/n
↓
3 Exponential Transform
V̂s,i = exp(−X̂i / λ̂s)    Under H₀ : X ~ Exp(λ) ⟹ V ~ U(0,1)    f̂V(v) & F̂V(v) estimated nonparametrically
↓
4 Covariance-Based Symmetry Measure
ℵ0.5 = Cov( fS 0.5(S),  FS(S) )
S = F(X)  ·  Handles uniform fS where ρ0.5 undefined  ·  Estimators: ℵ̂0.5,s (s = 1‒4)  &  ℵ̂C,0.5,s (s = 1,2)
H₀ (exponential) : ℵ0.5 = 0 H₁ (not exponential) : ℵ0.5 ≠ 0
↓
5 Empirical Likelihood Inference
ℓ̂n(ℵ0.5) → d χ²1    Reject H₀ if 0 ∉ CI    No pre-specified λ    CMC: ℓ̂C,n,s → χ²1
↓
★ λ̂₄ MLE-efficient
✓ λ̂₃, λ̂₄ outlier-robust
✓ CMC eliminates distortion
✓ Same variance ± distortion
✓ EL CIs validated
Path A — no distortion ψ(U) = 1 (§ 2)
